**Python Practical Questions**

### 1. Student Marks Analyzer

Create a program that:

* Accepts marks of 5 students.
* Stores them in a list.
* Displays:

  * Highest marks
  * Lowest marks
  * Average marks

---

### 2. Word Counter

Create a program that:

* Takes a sentence as input from the user.
* Counts the total number of words.
* Displays the longest word in the sentence.

**Example Input:**

```
Python is very easy to learn
```

---

### 3. Even and Odd Number Counter

Create a program that:

* Accepts 10 numbers from the user.
* Stores them in a list.
* Displays:

  * Total even numbers
  * Total odd numbers

---

### 4. Employee Data Using Dictionary

Create a program that:

* Creates a dictionary containing:

  * Employee ID
  * Employee Name
  * Salary
* Displays all details.
* Increases salary by 10% and displays the updated salary.

---

### 5. Mini Data Analysis Project

Create a program using Pandas that:

* Creates a DataFrame with:

  * Name
  * Age
  * Marks
* Displays:

  * All students
  * Students with marks greater than 75
  * Average marks of the class

**Sample Data:**

| Name  | Age | Marks |
| ----- | --- | ----- |
| Amit  | 20  | 85    |
| Priya | 21  | 72    |
| Rahul | 19  | 90    |
| Neha  | 22  | 68    |
| Arjun | 20  | 80    |
